Download settings

Below you will find the current download settings for the "aria2c" module.

Attention! You should change the settings when you are sure you know what you are doing, trying to change the settings without success may damage the process of downloading files using this application.

Current settings:

  • Default folder (Where you want to download files):


    • Default setting: % userprofile%\Desktop\WindowsBASE - "WindowsBASE" directory created on the desktop

    You can use the following variables:

    • % HOMEDRIVE% - system disk directory
    • % HOMEPATH% - your user account directory
    • % LOCALAPPDATA% - your user's application data directory
    • % TEMP% - temporary files directory
    • % PUBLIC% - directory of all system users
    • % OneDrive% - OneDrive sync folder directory

    Or enter a direct directory path, eg:

    C:\Users\User\Downloads\Downloads\

  • Download acceleration (Download acceleration option):

    • Enable download link band reservation
    Enable block download bundling
    Disable premature disk space reservation
    Enable keeping a stable HTTP connection
    Skip server certificate checking

    Default setting: On / On / On / Off / Off


  • Continue downloading:

    • Enable resuming file download from where it stopped

    Default setting: On


  • Verification:

    • Check the file for write errors and incompatible checksums

    Default setting: On


  • Other settings:

    • Enable saving of progress logs
    Limit download speed to 512 (KB/s):
    Use Legacy User-Agent Download Method:

    Default setting: On / Off / Off
